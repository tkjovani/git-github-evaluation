## Différence entre Git et GitHub

### Git
Git est un outil qui permet de gérer les versions d’un projet.
Il fonctionne en local sur l’ordinateur et permet de suivre les modifications des fichiers, de sauvegarder l’historique et de revenir à une version précédente en cas d’erreur.

### GitHub
GitHub est une plateforme en ligne qui permet d’héberger des projets utilisant Git.
Il sert à stocker le code sur Internet, à le partager et à travailler en équipe.

## Résumé
Git permet de gérer les versions d’un projet en local, tandis que GitHub permet de stocker et partager ce projet en ligne.
## À propos

Git est un système de contrôle de version qui permet de suivre l’évolution du code.
GitHub est une plateforme en ligne qui permet d’héberger des dépôts Git et de collaborer.
